# Redesign do index — Apple + Nike

## Direção
Criar uma home premium para o Hub de Ferramentas com:

- Clareza visual e hierarquia inspiradas na Apple.
- Energia, navegação por objetivo e cards editoriais inspirados na Nike.
- Identidade própria do Hub, sem copiar marca, assets ou layout literal.

## O que muda

1. Header limpo com navegação por categoria.
2. Hero claro com título grande, busca central e CTAs diretos.
3. Faixa de objetivos: orçar, comparar, precificar, buscar mercado, checklist e extensão.
4. Cards de ferramentas com visual de vitrine.
5. Manutenção dos links atuais e da lógica existente do index.
6. Responsivo para mobile com cards em coluna única.

## Arquivo principal

- `index.html`

## Critérios de aceite

- Busca continua filtrando os cards.
- Botão de busca rápida continua abrindo `scraper-dufrio.html` em modal.
- Ferramentas principais continuam apontando para os mesmos arquivos.
- Laboratório de scrapers permanece acessível.
- Layout funciona em desktop e celular.
